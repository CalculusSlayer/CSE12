Lab 1: Intro to Logic Simulation

-------------
DESCRIPTION

In this lab, the user can flip the input switches to see what the different outputs are. 
A light that is on has an output of 1 while a light that is off has an output
of 0.

-------------
Files

-
Lab1.lgi

This file contains the logic circuits for the lab. This is where users
can see the logic simulation.

--------------
Instructions

This program should be run with an MML program simulator. Flip the input switches to see
if the outputs are True or False.
