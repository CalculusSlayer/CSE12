Lab 5: Graphics

------------
DESCRIPTION

In this lab, a picture is printed in a small simulated display. For the test file,
diamond pattern with dark blue background is printed.

--------------
FILES

-
Lab5.asm

This file includes the assembly code of the lab.

-------------------
INSTRUCTIONS

In the MARS IDE, open up bitmap display in tools. Set the screen to 128 x 128 pixels.
Set the base address for display from memory map (0xFFFF0000). 
Finally, assemble and play the program.
